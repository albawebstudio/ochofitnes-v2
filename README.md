# Ocho Fitness

Ocho Fitness is a personal training and sports massage service dedicated to helping clients move better, get stronger, and stay pain-free through personalized fitness programs. The website emphasizes a tailored approach to fitness, offering in-person and online training options to meet individual goals and needs.

The Ocho Fitness website is powered by the following technologies:

- [AWS](https://aws.amazon.com/)
- [Nuxt 4.x](https://nuxt.com/)
- [Vue 3.x](https://vuejs.org/)
- [TailwindCss 3,x](https://v3.tailwindcss.com/)

based web application designed for professional fitness instructors and clients.



## Requirements

- [Node.js](https://nodejs.org/en/download/)
- A Node Package Manager:
  - [Npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
  - [Pnpm](https://pnpm.io/installation)
  - [Yarn](https://classic.yarnpkg.com/lang/en/docs/install/)
  - [Bun](https://bun.com/docs/installation)
- [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html)
